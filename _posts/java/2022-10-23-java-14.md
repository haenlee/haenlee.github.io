# 🚀 try-catch
---
💡 **try에서 에러가 났고 catch에서 메소드를 return 해버렸을 때 finally는 실행이 될까 안될까?**  
✔️ 실행이 된다. 
{: .notice--danger}

- try  
  예외가 발생할 수 있는 블록
- catch  
  예외처리를 명시할 수 있는 블록
- finally  
  가장 마지막에 실행이 필요한 것을 구현해 놓은 로직
